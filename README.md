# Kaspi.kz Ecosystem - PyQt6 Masterpiece Project

Профессиональное desktop-приложение на Python, имитирующее полноценную экожүйе Kaspi.kz. Проект разработан с соблюдением строгих инженерных стандартов, требований силлабуса и передовых практик Data Science.

## 🚀 Основные возможности

### 🛒 Умный Магазин (Marketplace 2.0)
- **Алгоритмическое ранжирование:** Поддержка категории "Танымал 🔥", которая автоматически формируется на основе данных о реальных продажах (NumPy Ranking).
- **Динамика цен:** Визуализация 30-дневной истории изменения цены товара с расчетом минимальной стоимости (Stochastic Simulation).
- **Бонусные науқандар:** Динамическая система кэшбэка (5% на акционные категории), мотивирующая пользователя.
- **Промокоды:** Система гибких скидок с валидацией через БД.
- **Социальный e-commerce:** Отзывы с фото, звездные рейтинги и визуальное распределение оценок (Rating Breakdown).

### 🏦 Цифровой Банкинг (FinTech Suite)
- **Multi-Account:** Поддержка Kaspi Gold, Kaspi Red, Рассрочки и **Kaspi Депозита**.
- **Умный Депозит:** Прогнозирование прибыли на 12 месяцев вперед с использованием NumPy (14% APR).
- **QR Экосистема:** 
  - *Scan to Pay:* Симулятор оплаты в торговых точках.
  - *My QR:* Генерация личного кода для приема P2P переводов.
- **Аналитика 360°:** 
  - *Pie Chart:* Структура трат по методам оплаты.
  - *Line Chart:* Хронологическая динамика расходов.
  - *Bar Chart:* Категориальный анализ.
- **Чеки и Экспорт:** Генерация текстовых квитанций и экспорт истории операций.

### 🤖 Интерактивные сервисы
- **Kaspi Guide:** Интеллектуальный чат-помощник, обрабатывающий запросы пользователя в реальном времени.
- **Система достижений:** Геймификация опыта через разблокировку ачивок (Achievement Service).
- **Персонализация:** Загрузка аватара, управление целями накоплений и "Недавно просмотренные" товары.

### 🔐 Безопасность и Стабильность
- **PIN Lock:** Защита входа в приложение через 4-значный код (Default: `0000`).
- **Admin Control:** Панель управления товарами и BI-аналитика продаж (Pass: `admin123`).
- **Reliable Data:** Thread-safe SQLite интеграция с автоматическими миграциями.

## 🛠 Технологический стек
- **Язык:** Python 3.12+
- **GUI:** PyQt6 (Advanced Layouts, Custom QPainter widgets, Animations, Dark Mode).
- **Data Science:** NumPy (Linear Regression, Stochastic Modeling, Distribution Analysis).
- **Database:** SQLite (Relational model, Atomic transactions).
- **QA:** Python Unittest (Commerce logic coverage).
- **i18n:** Полная локализация (Қазақша / Русский).

## 📂 Архитектура (Standard MVC)
Проект следует строгой архитектуре:
- `models/`: Сущности данных (Data Classes).
- `views/`: UI компоненты (Inheritance from `BaseView`).
- `controllers/`: Оркестрация навигации и состояний.
- `services/`: Сервисный слой для бизнес-логики и работы с данными.
